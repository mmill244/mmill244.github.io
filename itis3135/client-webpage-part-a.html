Project Webpage Requirements & Design

Project Overview

Purpose:

  • The purpose of this design document is to create a web-based application for an individual that will help promote their professional project. The project will be a personal project of the client and be focused on informing young women about the importance of women in STEM.

Intended Users of the Website:

  • The users of the website will be anyone interested in learning about women in STEM. This can include organizations that are interested in a lecture about the topic.

Overview of Content of Website:

  • The content on the website will consist of an introduction of my client, introduction of why she wanted to start this website/why the topic is important to her, and information that my client wants to share with the user.

Client Information

  1. Name: Maggie Wyatt
  2. Associations of Client: For this project, the content will be a personal project for the client. She would like to go around to different schools and gives speeches on the importance of women in STEM. So, there is no organization/institute/business directly involved with her.
  3. Client's Email and Phone Number: [private]

Pages of Website

Home Page

  1. Welcome
  2. The purpose of this page is to introduce the website to users.
  3. The users of this page will be clients of my client.
  4. The content of this page will include short explanations of what the website is aiming to do.
  5. Users will be asked to enter data if they want to sign up for a newsletter.
  6. This page will use buttons and have navigations to other pages.
  7. This page will use hyperlinks that will connect to other pages.

Get To Know The Client Page

  1. About Me
  2. The purpose of this page is to introduce my client to users of the website.
  3. The users of the page will be anyone who wants to learn more about my client.
  4. The content on this page will consists of information about my client.
  5. Users will not be asked to enter any data on this page.
  6. This page may include hyperlinks or buttons.

Why Women Should Go Into STEM

  1. Reasons to go into STEM
  2. The purpose of this page is to explain different reasons/benefits of STEM careers.
  3. The users of this page will be anyone interested in reasons to go into STEM.
  4. The content of this page will be reasons to go into a STEM career.
  5. Users will not be asked to enter data on this page.
  6. This page may include hyperlinks or buttons.

STEM Careers

  1. STEM Careers
  2. The purpose of this page is to give examples of STEM careers.
  3. The users of this page will be anyone interested in learning about STEM careers.
  4. The content of this page will be explanations of STEM careers.
  5. Users might be asked to enter data on this page.
  6. Thss page will include hyperlinks.

Fact Page

  1. Statistics About STEM
  2. The purpose of this page is to display facts about STEM careers.
  3. The users of this page will be anyone interested in interesting facts about STEM.
  4. The content of this page will be STEM facts.
  5. Users will not be asked to enter data on this page.
  6. I will know once I create the page if it needs any validators.

Dynamic Functionality

  • I want to use pictures on every page of the website.
  • The pictures will correspond with information on the pages and give a visual for users to look at.

Additional Information
